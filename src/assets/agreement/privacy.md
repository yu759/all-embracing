# 隐私政策 (Privacy Policy)

**版本更新日期：2026年1月13日**
**生效日期：2026年1月13日**

欢迎使用观测员平台。我们深知个人信息对您的重要性，并会尽全力保护您的隐私安全。

### 1. 我们收集的信息

为了提供基础的观测服务，我们会收集以下信息：

- **账号信息**：您的用户名、注册邮箱或手机号。
- **设备信息**：设备型号、操作系统版本、唯一设备标识符（IMEI/IDFA）、浏览器类型及 IP 地址。
- **地理位置**：当您提交观测记录时，我们需要获取您的**经纬度坐标**以确保数据的空间真实性。
- **操作日志**：包括您的登录时间、点击记录、观测数据上传记录。

### 2. 我们如何使用信息

- **验证身份**：确保只有合法的观测员能进入受限观测区域。
- **数据分析**：对观测点数据进行汇总，生成区域性研究报告。
- **安全保障**：监测异常登录行为，防止账号被非法冒用。

### 3. 信息共享与披露

我们不会将您的个人信息出售给第三方。但在以下情况除外：

- 获得您的明确授权后。
- 法律法规要求、司法机关指令或行政机关要求。
- 为了保护系统免受恶意攻击而必须进行的取证行为。

### 4. 数据安全

我们采用业界领先的加密技术（SSL/TLS）传输数据，并对核心敏感信息（如坐标偏移量）进行加密存储。请注意，互联网并非百分之百安全，建议您定期更换密码。

### 5. 您的权利

- **访问与更正**：您可以在个人中心随时查阅或修改您的信息。
- **撤回同意**：您可以通过关闭定位权限或注销账号来撤回您的观测授权。
- **数据下载**：您可以申请导出您的历史观测报告副本。
